# Basic-OCR-Engine-for-seq2seq-Digit-Recognition

This project implements a basic OCR engine for handwritten digit sequence recognition using PyTorch and the EMNIST dataset.

## Technologies
- Python
- PyTorch
- NumPy
- Torchvision

## Dataset
EMNIST

## Results
The model is trained to recognize digit sequences using a seq2seq approach.
